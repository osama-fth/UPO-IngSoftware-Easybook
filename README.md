# Gestionale per Biblioteca

## Descrizione
Applicazione per la gestione del catalogo libri, dei prestiti e delle multe di una biblioteca. Interfacce per bibliotecario e App utente; invio notifiche via email e job automatici per scadenze e sanzioni.
